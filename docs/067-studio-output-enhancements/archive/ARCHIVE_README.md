# Archive policy

This folder contains superseded immutable documentation snapshots for `067-studio-output-enhancements`.

## Rules

- Files are moved here only after a newer version is created in the work package root.
- Archived files must not be edited.
- The latest active spec and plan versions remain in `docs/067-studio-output-enhancements/`.
- Archived filenames are preserved exactly so version history stays traceable.
